# Earth's Exhale

A responsive indoor plant shopping website built with Next.js and Tailwind CSS.

## Live Demo
[https://earth-exhale-93cjse978-aare-harinis-projects.vercel.app](https://earth-exhale-93cjse978-aare-harinis-projects.vercel.app)

## Tech Stack
- Next.js (App Router)
- Tailwind CSS
- React

## Setup

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)

## Features
- Responsive design across mobile, tablet, and desktop
- Modern UI with dark green theme
- Reusable React components
- Smooth animations and hover effects
- Mobile-friendly hamburger menu
